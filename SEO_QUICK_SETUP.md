## 🎯 3-Step Setup (15 Minutes)

### Step 1: Deploy Your Website (5 min)

**You MUST deploy first!**

```bash
# Follow QUICK_DEPLOY.md
git push origin main
# Connect to Vercel
# Done!
```

**Result**: Your site will be live at `https://bestold.vercel.app`

---

### Step 2: Submit to Google (5 min)

#### A. Go to Google Search Console
**URL**: https://search.google.com/search-console

#### B. Add Your Site
1. Click "Add property"
2. Enter: `https://bestold.vercel.app`
3. Click "Continue"

#### C. Verify Ownership (Choose ONE method)

**Method 1: HTML File** (Easiest)
1. Download verification file
2. Put in `/public/` folder
3. Deploy again
4. Click "Verify"

**Method 2: Meta Tag**
1. Copy the meta tag
2. Add to `index.html` (already has space for it)
3. Deploy again
4. Click "Verify"

---

### Step 3: Submit Sitemap (5 min)

#### A. Generate Sitemap
1. Go to: `https://bestold.vercel.app/admin/seo`
2. Click "Download Sitemap"
3. Save `sitemap.xml`

#### B. Upload Sitemap
1. Put `sitemap.xml` in `/public/` folder
2. Deploy again

#### C. Submit to Google
1. In Google Search Console
2. Go to "Sitemaps" (left menu)
3. Enter: `sitemap.xml`
4. Click "Submit"

**Done!** ✅

## 🆘 Troubleshooting

### "My site still doesn't show on Google"

**Check**:
1. ✅ Is site deployed? (not localhost)
2. ✅ Did you submit to Search Console?
3. ✅ Did you submit sitemap?
4. ✅ Has it been at least 3 days?

**If yes to all**: Wait 1-2 more weeks. Google is slow!

### "I submitted but nothing happens"

**Solutions**:
1. Check Search Console for errors
2. Request indexing manually:
   - Go to URL Inspection
   - Enter your homepage URL
   - Click "Request Indexing"
3. Share on social media (creates backlinks)
4. Wait 1-2 weeks

### "Only homepage is indexed"

**This is normal!** Google indexes pages gradually.

**Speed it up**:
1. Submit sitemap (if not done)
2. Share product links on social media
3. Add internal links between pages
4. Wait 1-2 weeks
