# Quick Start: Sell Your Phone Feature

## What You Get

✅ **"Sell Your Phone" button** on homepage
✅ **Easy form** with brand, model, condition, age selection
✅ **Image upload** for front and back of phone
✅ **WhatsApp integration** - submissions sent directly to your WhatsApp
✅ **Admin panel** to customize all options
✅ **Submission tracking** - all data saved in database

## For Customers

### How to Sell a Phone

1. **Find the Button**
   - Go to homepage
   - Look for **"Sell Your Phone"** button below the search bar

2. **Fill the Form**
   - Select your phone brand (e.g., Apple)
   - Select model (e.g., iPhone 15 Pro)
   - Choose condition (Like New, Good, Fair, Poor)
   - Select how old it is (Less than 6 months, 1-2 years, etc.)

3. **Upload Photos**
   - Click to upload front image
   - Click to upload back image
   - Max 5MB per image

4. **Your Contact Info**
   - Enter your name (optional)
   - Enter your phone number (required)
   - Enter your email (optional)

5. **Submit**
   - Click **"Submit & Send to WhatsApp"**
   - Your details will be sent to the store owner
   - You'll see a success message

## For Admin

### Quick Setup (First Time)

1. **Set Your WhatsApp Number**
   - Go to **Admin Panel → Sell Phone → Settings tab**
   - Enter country code: `+91`
   - Enter number: `8167865019`
   - Click **"Save Settings"**

2. **Add Phone Models** (Optional)
   - Go to **Models tab**
   - Click **"Add Model"**
   - Select brand (e.g., Apple)
   - Enter model name (e.g., iPhone 16 Pro)
   - Click **"Create Model"**

3. **Test It**
   - Go to homepage
   - Click **"Sell Your Phone"**
   - Fill the form and submit
   - Check your WhatsApp for the message

### Managing Options

#### Add a New Brand
1. Admin Panel → Sell Phone → **Brands tab**
2. Click **"Add Brand"**
3. Enter name (e.g., "Huawei")
4. Set display order (e.g., 110)
5. Toggle **Active** on
6. Click **"Create Brand"**

#### Add a New Model
1. Admin Panel → Sell Phone → **Models tab**
2. Click **"Add Model"**
3. Select brand
4. Enter model name (e.g., "P60 Pro")
5. Click **"Create Model"**

#### Edit Conditions
1. Admin Panel → Sell Phone → **Conditions tab**
2. Click pencil icon next to condition
3. Update name or description
4. Click **"Save Changes"**

#### Change Age Options
1. Admin Panel → Sell Phone → **Age tab**
2. Click pencil icon to edit
3. Or click **"Add Age Option"** for new one
4. Update and save

## Tips

### For Best Results

✅ **Keep options simple** - Don't add too many brands/models
✅ **Update regularly** - Add new phone models as they release
✅ **Clear descriptions** - Help customers choose the right condition
✅ **Test WhatsApp** - Submit a test phone to verify it works
✅ **Quick responses** - Reply to WhatsApp messages promptly

### Common Tasks

**Deactivate a brand instead of deleting**:
- Go to Brands tab
- Toggle the switch to off
- Brand won't show in customer form

**Reorder options**:
- Edit the item
- Change display_order number
- Lower numbers appear first

**Change WhatsApp number**:
- Go to Settings tab
- Update number
- Save changes
- Test with a submission

## Troubleshooting

### Button Not Showing
- Check if you're on the homepage
- Refresh the page
- Clear browser cache

### WhatsApp Not Opening
- Verify WhatsApp number in Settings tab
- Check country code is correct
- Ensure WhatsApp is installed

### Images Not Uploading
- Check image size (max 5MB)
- Use JPG or PNG format
- Try compressing the image

### Models Not Showing
- Make sure you selected a brand first
- Check if brand has active models
- Verify models are marked as active
